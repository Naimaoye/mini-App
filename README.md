# mini-App
A mini react Application used for todoList
## Technologies

- ESLint
- Babel
- Webpack4

## Requirements and Installation

To install and run this project you would need to have listed stack installed:

- npm

- To run:

```sh
git clone <https://github.com/Naimaoye/mini-App.git>
cd mini-App
npm install
npm run start
```
